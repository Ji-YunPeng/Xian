主要功能：
主页
鼠标滑过三级菜单，
轮播图，
一楼数据渲染的商品选项卡，选项卡的第一页为固定数据，第二页的商品列表可以点击进入商品详情
楼层 可以点击到对应楼层，滚轮到一楼后楼层才会出现超过一楼后淡出

购物车页面：商品数量可以更改点击删除可以删除该商品的记录

商品列表页面：有分页随数据的改变而改变 可以添加商品，可以点击进入商品详情登录之后才能添加购物车

商品详情页面：基本的放大镜功能登录之后才能添加购物车 可以改变商品件数

注册页面：使用正则来判断数据，使用localstorage来保存数据

登录页面：如果登录成功跳转首页。如果登录成功且点击了记住账号，下一次注销后跳转到登录页面时账号密码已经存在，可以直接登录。如果登录成功时记住账号为空，下一次注销后跳转到登录页面时账号密码要重新输入。

点击页面：
点击Logo可以跳转到主页
点击第三级菜单的商品列表可以跳转到商品列表
点击右上角的登录与注册，可以分别跳转到登录与注册页面，登录之后右上角的登录与注册会变成注销点，击注销会跳转到登录页面并且成为没有登录状态需要重新登录
在未登录状态可以浏览网页与商品，但是无法添加购物车
在未登录状态下点击加入购物车会出现弹窗点击确定跳转登录页面
在未登录状态下购物车没有商品的之前购买的商品未数据渲染
点击右上角的我的订单可跳转到购物车
点击我的购物车悬浮框可跳转的购物车
在登录的情况下购物车悬浮框会显示当前购物车的商品总量

所有数据都是由localstorage来保存请使用时清空数据
本网页的环境为node  端口为8181
